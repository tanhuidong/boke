# SSL证书申请

### 申请流程
    1. 域名准备好，解析到服务器上
    2. 服务器443端口开始，https需要用到443端口
    3. 登录阿里云购买20条/年免费SSL证书
    4. 申请前先确认好是否需要csr文件（公钥），不需要申请时在阿里云选择自动生成csr文件即可，需要则自行手动生成。
        - 选择一：再阿里云手动生成
        - 选择二：第三方工具生成（这里选择天威诚信SSL证书生成工具：https://www.itrus.cn/service_view_311.html）
        - 选择三：在服务器中手动生成
    5. 自动生成（生成证书后，将无法再获取csr文件了，再想要只能重签证书，选择手动生成）。
        - 填写基本信息，提交申请
        - 选择自动生成csr证书
        - 验证信息完毕后提交审核
        - 等待签发完毕后下载证书文件可以密码（不同服务器证书格式不同安装方法也不同，阿里云中提供详细教程）
    6. 手动生成。
        - 填写基本信息，提交申请
        - 安装好天威工具，填写基本信息，生成csr文件。生成后会有两个文件，一个是公钥csr，一个是私钥key。
        - 阿里云提交申请时将公钥粘贴进去
        - 信息验证完毕后提交审核
        - 证书签发完毕会返回一个pem文件
        - 将pem和key文件在天威工具中合并就会得到证书文件
    7. 在服务器上将证书安装完毕
